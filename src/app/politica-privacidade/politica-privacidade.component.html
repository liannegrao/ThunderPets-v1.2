Política de Privacidade - LGPD

1. Introdução

A ThunderPets ("nós", "nosso" ou "empresa") está comprometida em proteger a privacidade e os dados pessoais dos usuários da nossa plataforma de adoção terapêutica de pets. Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos suas informações pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).

2. Dados Pessoais Coletados

Coletamos os seguintes tipos de dados pessoais:

  • Dados de identificação: Nome completo, CPF, data de nascimento
  • Dados de contato: Endereço de e-mail, número de telefone
  • Dados de localização: Cidade e estado de residência
  • Dados comportamentais: Preferências de adoção, histórico de navegação na plataforma
  • Dados sensíveis: Informações sobre saúde emocional (níveis de depressão, ansiedade, solidão) para matching terapêutico
  • Dados de pets: Informações sobre animais cadastrados (espécie, raça, idade, condições especiais)

3. Finalidade do Tratamento dos Dados

Utilizamos seus dados pessoais para:

  • Fornecer serviços de matching terapêutico entre adotantes e pets adequados
  • Gerenciar processos de adoção e acompanhamento pós-adoção
  • Manter comunicação sobre adoções em andamento
  • Melhorar nossos serviços através de análise de dados
  • Cumprir obrigações legais e regulatórias
  • Prevenir fraudes e garantir segurança da plataforma

4. Compartilhamento de Dados

Seus dados pessoais podem ser compartilhados com:

  • Doadores e adotantes: Informações necessárias para o processo de adoção
  • Mediadores: Profissionais que auxiliam no processo terapêutico
  • Provedores de serviços: Cloudinary (para armazenamento de imagens), apenas dados técnicos necessários
  • Autoridades legais: Quando exigido por lei ou ordem judicial

Não vendemos ou alugamos dados pessoais para terceiros.

5. Seus Direitos sob a LGPD

De acordo com a LGPD, você tem os seguintes direitos:

  • Confirmação: Saber se tratamos seus dados pessoais
  • Acesso: Obter cópia dos dados pessoais que mantemos
  • Correção: Solicitar correção de dados incompletos ou inexatos
  • Anonimização: Solicitar anonimização de dados desnecessários
  • Portabilidade: Solicitar transferência de dados para outro fornecedor
  • Eliminação: Solicitar exclusão de dados quando não houver base legal para tratamento
  • Informação: Saber com quem compartilhamos seus dados
  • Revogação: Revogar consentimento para tratamento de dados sensíveis

6. Segurança dos Dados

Implementamos medidas técnicas e organizacionais para proteger seus dados pessoais:

  • Criptografia de dados em trânsito e armazenamento
  • Controle de acesso restrito aos dados
  • Auditorias regulares de segurança
  • Backup seguro e recuperação de desastres
  • Treinamento da equipe sobre proteção de dados

7. Retenção de Dados

Retemos seus dados pessoais apenas pelo tempo necessário para cumprir as finalidades descritas, ou conforme exigido por lei:

  • Dados de usuários ativos: Enquanto mantiver conta ativa
  • Dados de adoções: 5 anos após conclusão, para garantia legal
  • Dados de comunicação: 2 anos para resolução de disputas
  • Dados anonimizados: Indefinidamente para análise estatística

8. Cookies e Tecnologias Similares

Utilizamos cookies para melhorar sua experiência na plataforma:

  • Cookies essenciais: Para funcionamento básico da plataforma
  • Cookies de preferência: Para lembrar suas configurações
  • Cookies analíticos: Para entender como a plataforma é usada

Você pode gerenciar cookies através das configurações do seu navegador.

9. Menores de Idade

A plataforma é destinada a maiores de 18 anos. Não coletamos intencionalmente dados de menores de idade. Se descobrirmos que coletamos dados de menor, eles serão imediatamente excluídos.

10. Alterações nesta Política

Podemos atualizar esta política periodicamente. Alterações significativas serão comunicadas através de e-mail ou aviso na plataforma. O uso continuado após alterações constitui aceitação da nova política.
